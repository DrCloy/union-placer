# Commit Convention

Conventional Commits 형식. 영어로 작성.

```
<type>: <subject>

<body> (optional)
```

- subject: 50자 이내, 동사 원형, 마침표 없음
- body: subject만으로 부족할 때만, bullet(-) 사용

## Types

`feat` `fix` `docs` `style` `refactor` `test` `chore`

## 예시

```
feat: add block rotation logic
fix: correct cell counting on character toggle (#45)
chore: add ESLint import layer rules
```

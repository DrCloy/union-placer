# NEXON Open API 서비스 이용약관

> 본 약관은 NEXON Open API 서비스와 관련하여 '넥슨 통합 서비스 이용약관' 및 기타 회사가 정한 약관 및 운영정책에 우선적으로 적용되며, 본 약관에 기재되지 않은 내용은 기타 회사가 정한 약관 및 운영정책에 따릅니다.

---

## 제 1 조 (목적)

NEXON Open API (넥슨 Open API) 서비스 이용약관(이하 '본 약관')은 ㈜넥슨코리아(이하 '회사')와 이용자 간에 회사가 제공하는 API 서비스의 권리, 의무 및 책임에 관한 제반 사항과 기타 필요한 사항을 규정함을 목적으로 합니다.

---

## 제 2 조 (용어의 정의)

이 약관에서 사용하는 용어의 정의는 다음과 같습니다.

1. **API**란 Application Programming Interface의 약자로 이용자가 본인의 프로그램 등에서 회사가 제공하는 기술을 사용하기 위한 인터페이스를 말하며, **API 서비스**란 회사가 이와 같은 API를 이용자에게 제공하는 것을 말합니다.
2. **이용자**란 넥슨 서비스 이용약관에 따라 가입 신청 및 이용 계약 체결을 완료한 회원 중, 본 약관에 따라 회사와 이용계약을 체결하고 회사가 제공하는 API 서비스를 이용하는 자를 말합니다.
3. **API Key**란 API 서비스 사용하기 위해서 회사가 이용자에게 부여하는 고유한 인증 값을 말합니다.

---

## 제 3 조 (약관의 효력, 명시 및 개정)

1. 본 약관은 이용자가 API 서비스 이용을 위하여 동의함으로써 효력이 발생하게 됩니다.
2. 회사는 필요하다고 인정되는 경우, 관련법령에 위배되지 않는 범위 내에서 본 약관을 개정할 수 있습니다. 회사는 약관이 변경되는 경우에 변경된 약관의 내용과 시행일을 정하여, 그 시행일로부터 **7일전** 홈페이지(https://openapi.nexon.com)에 온라인으로 공시합니다. 다만, 이용자에게 불리하게 약관 내용을 변경하는 경우에는 시행일로부터 **30일전** 홈페이지 온라인으로 공시하거나, 이용자가 넥슨 서비스 회원 가입 시 등록한 e-mail로 전송하는 방법으로 이용자에게 고지합니다. 변경된 약관은 공시하거나 고지한 시행일로부터 효력이 발생합니다.
3. 이용자는 개정된 약관에 거부할 권리가 있으며, 개정 약관의 적용에 동의하지 않을 경우 서비스 이용을 중단할 수 있습니다. 변경된 이용약관의 시행일 이후에도 서비스를 계속 이용하는 경우에는 변경된 약관에 동의한 것으로 간주된다는 내용을 제2항에 따라 함께 공시 또는 고지하고, 이용자가 시행일 이후 서비스를 이용하는 경우 개정된 약관에 동의한 것으로 봅니다.

---

## 제 4 조 (API 서비스의 제공)

1. 이용자는 NEXON Open API에서 제공하는 오픈 API를 통해 만들어진 응용프로그램 및 웹사이트 등(모바일 웹, 모바일 어플리케이션을 모두 포함하며 이하 '프로그램 등'이라 함)을 통해 API를 제공하는 넥슨게임의 캐릭터, 서버 정보 등의 메타데이터 및 유저들간의 매치 기록이나 유저들의 게임 진행에 대한 기록 등 넥슨게임 서비스의 컨텐츠 내용에 대한 데이터를 검색할 수 있습니다.
2. 회사는 API 서비스의 범위, 이용가능 시간, 이용 가능 횟수 등 API 서비스의 제공 조건을 정하여 서비스 홈페이지에 통해 공지하며, 이용자는 그와 같이 정해진 제공 조건에 따라서 API 서비스를 이용할 수 있습니다.
3. 이용자는 회사에서 제공하는 API를 활용하여 개발한 어플리케이션 등을 Open API 홈페이지에 등록할 수 있으며, 회사는 이용자가 등록한 어플리케이션 등에 대해 다른 회원에게 공개될 수도 있음을 동의한 것으로 간주하고 별도 의사 확인 없이 등록된 어플리케이션 등을 개발자 홈페이지 내 공개할 수 있습니다.
4. 회사의 API 서비스 제공은 어떠한 경우에도 회사와 이용자 간 제휴 관계의 성립을 의미하지 않습니다.

---

## 제 5 조 (이용자의 의무)

1. 이용자는 자신이 발급 받은 API Key를 타인에게 제공, 공개하거나 공유할 수 없습니다. 단, 일부 API의 경우 API Key 소유자가 공개를 허용할 경우 API Key를 타인에게 제공할 수 있으며, 타인에게 제공 가능한 API Key는 Open API 공지사항 또는 내 애플리케이션 페이지를 통해 확인할 수 있습니다.
2. 이용자는 자신이 발급 받은 API Key가 타인에 의하여 도용 당하지 않도록 관리하여야 하며, 타인이 발급받은 API Key를 이용해서는 안 됩니다. 단, 일부 API의 경우 API Key 소유자가 공개를 허용할 경우 API Key를 타인에게 제공할 수 있으며, 타인에게 제공 가능한 API Key는 Open API 공지사항 또는 내 애플리케이션 페이지를 통해 확인할 수 있습니다.
3. 이용자는 프로그램 등을 제공하면서 그 사용자들로부터 API 서비스 이용에 대한 대가 및 프로그램 등의 이용에 대한 대가(금전, 정보 제공 등 일체의 유·무형의 대가)를 제공 받을 시 홈페이지 내 안내된 내용에 따라야 하며, 게임 IP 사용 가이드 등 회사가 정한 정책 및 관련 법령을 위반하여서는 안됩니다.
4. 이용자는 API 서비스를 이용하면서 회사의 임직원이나 관계자를 사칭하거나 회사의 브랜드나 서비스를 모방, 도용하는 행위를 해서는 안 됩니다.
5. 이용자는 회사의 사전 동의 없이 API 서비스의 결과 데이터(넥슨게임의 캐릭터, 서버 정보 등의 메타데이터 및 유저들간의 매치 기록이나 유저들의 게임 진행에 대한 기록 등 넥슨게임 서비스의 컨텐츠 내용에 대한 데이터를 포함)를 본 약관에서 허용한 범위를 넘어서서 무단으로 복제, 저장, 가공, 배포하거나 프로그램 등을 통해 API 서비스를 다시 제 3자에게 제공하는 행위는 할 수 없습니다.
6. 이용자는 회사 혹은 제 3자의 명예를 훼손하거나 지식재산권을 포함한 일체의 권리를 침해하는 행위를 해서는 안 됩니다.
7. 이용자는 다음과 같은 내용을 포함하는 프로그램 등에는 API 서비스를 적용하면 안 됩니다.
   - 정보통신망법상 불법적인 내용(법 제44조의7 제1항의 각호의 내용)
   - 청소년보호법상 청소년유해매체물에 해당하는 선정적, 폭력적, 혐오적, 반사회적, 비도덕적, 불법적인 내용
   - 회사 혹은 제3자의 저작권 등 지식재산권을 침해하는 내용
   - 회사 혹은 제3자의 명예를 훼손하거나 모욕하는 내용
   - 타인의 초상권, 성명권 등 인격권을 침해하는 내용
   - 기타 범죄를 교사하거나 현행법에 위반되는 내용
8. 이용자는 회사의 API 서비스를 제공하는 시스템(서버 등)에 자동화 프로그램을 이용하여 접속하거나 접근권한을 넘어서는 접속을 시도하거나 주기적/지속적인 접속을 시도하여 과부하를 발생시키는 등의 방식으로 회사의 정보통신망 및 정보통신시스템에 접근해서는 안 됩니다.
9. 이용자는 프로그램 등을 통해 악성코드를 배포해서는 안 됩니다.
10. 이용자는 본 약관에서 규정하는 사항과 기타 회사가 정하여 공지한 제반 사항 및 관련 법령을 준수하여야 하며, 기타 회사의 업무에 방해가 되는 행위 및 회사의 명예를 손상시키는 행위를 해서는 안 됩니다.

---

## 제 6 조 (지식재산권의 귀속)

1. 회사가 제공하는 API 서비스 및 관련하여 회사가 제작한 프로그램, 디자인, 상표 등의 지식재산권 등 제반 권리는 **회사에 귀속**되며, API 서비스를 이용한 결과 데이터의 저작권 등 제반 권리는 회사 또는 제3자에게 있습니다.
2. 회사의 이용자의 API 서비스 이용에 대한 승낙은 API 서비스 및 결과 데이터에 대한 본 약관에 따른 **제한된 사용권의 부여**만을 의미할 뿐, 어떠한 경우에도 (그와 같은 제한된 사용권의 부여를 넘어선) 이용자의 API 서비스와 관련된 지식재산권 혹은 관련 권리의 취득 혹은 결과 데이터에 대한 권리의 취득을 의미하지는 않습니다.
3. 회사의 API 서비스 이용 승낙은 회사가 이용자에게 등록상표 또는 저작물에 대하여 자유로운 이용권을 부여하는 것으로 해석될 수 없습니다. 이용자는 NEXON Open API 운영정책 등을 통해 회사가 허락한 경우 및 그 범위에 한하여만 이를 사용할 수 있으며, 사용이 가능한 경우에도 게임 IP사용가이드 및 운영정책 등에 따른 이용기준을 준수하여야 하고 임의로 이를 복제, 변형하거나 훼손하여서는 안 됩니다.
4. 이용자는 프로그램 등 및 API 서비스를 이용한 결과 데이터에 회사가 별도로 정하는 방법에 따라 **"NEXON Open API"를 이용한 것임을 명시**해야 하며, 회사의 브랜드를 모방, 도용하여서는 안 됩니다.
5. 회사가 제공하는 API 서비스 및 결과 데이터에 포함된 콘텐츠가 이용자 창작 콘텐츠로서 저작권 혹은 기타 지식재산권이 그 이용자에게 정당하게 귀속하는 경우, 해당 이용자는 언제든지 1:1 문의하기(help_openapi@nexon.co.kr)를 통해 해당 콘텐츠에 대해 결과 데이터 제외 또는 비공개를 요청할 수 있습니다.
6. 이용자는 API 서비스를 이용하면서 얻은 일체의 데이터 및 저작물 등을 회사의 승낙 없이 영리 목적으로 이용하거나 회사가 허락한 경우 및 그 범위를 초과하여 제3자에게 이용하도록 할 수 없습니다.

---

## 제 7 조 (서비스의 제한)

1. 회사는 원활한 API 서비스 제공을 위해 운영상 또는 기술상 필요에 따라 API 서비스를 변경, 일시중지 또는 중단할 수 있으며, 특정 범위를 분할하거나 별도의 이용가능 대상, 시간 또는 횟수를 지정 및 변경할 수 있습니다. 회사는 변경 전에 해당 내용을 홈페이지에 공지하며, 사전에 공지할 수 없는 부득이한 사유가 있는 경우에는 사후에 그 사유와 함께 공지합니다.
2. 이용자가 발급된 API Key를 **1년 이상 사용하지 않은 경우**, 회사는 사전 안내 후 해당 API Key를 삭제할 수 있습니다.
3. 회사는 이용자가 본 약관을 위반한 경우, API 서비스의 제공을 중단하거나, 필요할 경우 이용계약을 해지할 수도 있습니다.

---

## 제 8 조 (면책사항)

1. 회사는 무료로 제공되는 API 서비스의 결과의 정확성, 적합성, 신뢰성, 정당성, 적법성, 지속성 등을 보증하지는 않으며, API의 사용 및 사용불능으로 인하여 이용자에게 발생한 손해에 대해서도 책임을 지지 않습니다.
2. 회사는 이용자가 약관, 운영정책, 게임 IP사용가이드, 그 외 서비스 이용 방법 및 이용 기준 등을 준수하지 않아 발생한 이용자 및 제3자의 손해에 대하여 책임을 부담하지 않습니다.
3. 회사는 이용자 상호 간 및 이용자와 제3자 간(정부 기관 포함)에 API서비스를 매개로 발생한 분쟁에 대하여 개입할 의무가 없으며, 해당 분쟁에서의 손해에 대한 배상 책임 또한 지지 않습니다. 만약 이용자와 분쟁중인 제3자가 회사를 상대로 이의를 제기할 경우 이용자는 자신의 비용과 책임으로 문제를 직접 해결하고 회사를 면책시켜야 하며 회사에 발생한 손해를 배상하여야 합니다.

---

## 제 9 조 (개인정보보호, 로그정보의 보관 및 활용)

1. 회사는 API 서비스를 운영함에 있어 이용자가 제공하는 개인정보의 수집 및 이용에 대해서는 본 이용약관과 별도로 게시된 **"개인정보처리 방침"**이 적용됩니다.
2. 회사는 API 서비스를 제공함에 따라 발생하는 로그정보(이용자가 API로 전송한 정보)를 보관할 수 있으며 API 서비스의 향상을 위해 활용할 수 있습니다.

---

## 제 10 조 (양도의 금지)

이용자는 본 약관에 의해 체결된 계약상 권리 및 의무 등 일체의 계약상 지위를 회사의 동의 없이 제3자에게 양도할 수 없습니다.

---

## 제 11 조 (이용 계약의 해지)

1. 이용자가 이용계약을 해지하고자 하는 때에는 회사가 정한 절차에 따라 이용 계약을 해지할 수 있습니다.
2. 회사는 이용자가 약관을 준수하지 않는 경우 API서비스 이용계약을 해지하고 이용자의 API서비스 이용을 중단시킬 수 있습니다.
3. 이용계약이 해지되거나 API 서비스가 중단된 경우 이용자는 프로그램 등에서 API 사용을 즉시 중단하고 API 및 API 서비스를 이용한 결과 데이터 일체를 삭제하여야 합니다.

---

## 제 12 조 (준거법 및 관할 법원)

본 약관의 준거법은 **대한민국의 법**으로 하며, API 서비스 이용으로 회사와 이용자 간에 발생한 분쟁에 대해 소송이 제기되는 경우 민사소송법에 따른 법원이 관할법원이 됩니다.

---

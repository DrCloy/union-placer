# 화면 흐름도 (User Flow)

## 1. 전체 흐름

```mermaid
flowchart TD
    subgraph 진입
        A[서비스 접속] --> B{블록 입력 방식 선택}
    end

    subgraph 블록입력["블록 입력"]
        B -->|넥슨 로그인| C1[OAuth 로그인]
        B -->|캐릭터 검색| C2[닉네임 입력]
        B -->|수동 입력| C3[블록 개수 직접 입력]

        C1 --> D1[월드 선택]
        D1 --> E1[캐릭터 목록 표시]

        C2 --> E2[유니온 캐릭터 목록 표시]

        E1 --> F[캐릭터 선택]
        E2 --> F

        F --> G[블록 카운팅 표시]
        C3 --> G
    end

    subgraph 설정
        G --> H[목표 칸 수 입력]
        H --> I[우선순위 설정]
        I -->|프리셋| I1[사냥/보스 선택]
        I -->|커스텀| I2[영역별 순위 설정]
        I1 --> J[설정 완료]
        I2 --> J
    end

    subgraph 탐색
        J --> K[배치 시작]
        K --> L{탐색 중}
        L -->|완료| M[결과 표시]
        L -->|중단| N[중단됨]
        L -->|0순위 불가| O[차선책 + 부족 영역 안내]
    end

    subgraph 결과
        M --> P[유니온 판 시각화]
        O --> P
        N --> Q{다음 액션}
        P --> Q
        Q -->|재설정| H
        Q -->|블록 변경| F
        Q -->|처음부터| B
    end
2. 화면 구성화면주요 요소진입방식 선택 버튼 (로그인 / 검색 / 수동)블록 입력캐릭터 카드 목록, 선택/해제, 등급 필터, 블록 카운팅설정목표 칸 수 입력, 우선순위 설정 (프리셋/커스텀)결과유니온 판 시각화, 영역별 칸 수, 재설정 버튼3. 화면별 상세3.1 진입 화면
서비스 소개 (한 줄)
블록 입력 방식 3가지 선택

넥슨 로그인 (OAuth)
캐릭터 검색 (닉네임)
수동 입력


3.2 블록 입력 화면3.2.1 캐릭터 목록 (API 조회 시)
카드 형태로 캐릭터 표시
카드 정보: 닉네임, 직업, 레벨, 등급, 블록 모양 미리보기
클릭으로 선택/해제
필터: 등급별 (B/A/S/SS/SSS)
버튼: 전체 선택, 전체 해제
메이플M 슬롯: 별도 1명 추가
3.2.2 수동 입력 시
등급 × 블록 모양별 개수 입력
총 칸 수 실시간 표시
3.2.3 공통
블록 카운팅 요약 표시
다음 단계 버튼
3.3 설정 화면3.3.1 목표 칸 수
숫자 입력 필드
버튼: 0으로 초기화, 최대로 채우기
제한: 블록 총합 이하
3.3.2 우선순위
탭: 프리셋 / 커스텀
프리셋: 사냥, 보스 선택
커스텀: 칸반 형태 드래그 앤 드롭

0순위 (필수)
1순위 이하 (중요도 순)
미지정


3.4 결과 화면
유니온 판 시각화 (22 × 20)
블록별 색상 + 테두리 구분
영역별 채워진 칸 수 표시
버튼: 재설정, 블록 변경, 처음부터
```

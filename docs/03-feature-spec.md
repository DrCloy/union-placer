# 기능 명세서

## F1. 블록 입력

### F1.1 API 조회 - 방식 A: 넥슨 로그인 (OAuth)

| 항목 | 내용                                                        |
| ---- | ----------------------------------------------------------- |
| 입력 | 넥슨 계정 OAuth 로그인                                      |
| 출력 | 월드 목록 → 월드 선택 → 캐릭터 목록 (직업, 레벨, 블록 모양) |
| 예외 | 로그인 실패 시: 에러 표시 + 재시도 또는 다른 방식 안내      |

### F1.2 API 조회 - 방식 B: 캐릭터 검색

| 항목 | 내용                                                        |
| ---- | ----------------------------------------------------------- |
| 입력 | 캐릭터 닉네임                                               |
| 출력 | 해당 유니온 캐릭터 목록 (직업, 레벨, 블록 모양)             |
| 예외 | 닉네임 조회 실패 시: 에러 표시 + 재시도 또는 수동 입력 안내 |

### F1.3 캐릭터 선택

| 항목    | 내용                                     |
| ------- | ---------------------------------------- |
| 입력    | 캐릭터 목록                              |
| 출력    | 선택된 캐릭터들의 블록 카운팅            |
| 동작    | 카드 형태로 표시, 클릭으로 선택/해제     |
| 제한    | 유니온 등급에 따른 최대 인원 (최대 45명) |
| 버튼    | 전체 선택 (레벨 높은 순), 전체 해제      |
| 필터    | 등급별 (B/A/S/SS/SSS)                    |
| 메이플M | 별도 슬롯 1명 추가 가능                  |

### F1.4 수동 입력

| 항목 | 내용                    |
| ---- | ----------------------- |
| 입력 | 블록 종류별 개수        |
| 출력 | 블록 카운팅 + 총 칸 수  |
| 동작 | 등급 × 직업별 개수 입력 |

## F3. 우선순위 설정

| 항목 | 내용                         |
| ---- | ---------------------------- |
| 입력 | 프리셋 선택 또는 커스텀 설정 |
| 출력 | 영역별 우선순위              |

### 프리셋

| 프리셋 | 우선순위                                    |
| ------ | ------------------------------------------- |
| 사냥   | 획득 경험치 > 크리티컬 데미지 > 일반 데미지 |
| 보스   | 크리티컬 데미지 > 보스 데미지 > 방어율 무시 |

### 커스텀

| 항목          | 내용                          |
| ------------- | ----------------------------- |
| UI            | 칸반 형태, 드래그로 순서 변경 |
| 0순위         | 필수 영역 (무조건 채움)       |
| 1순위 이하    | 중요도 순서                   |
| 미지정 (외부) | 배치 안 함                    |
| 미지정 (내부) | 연결 필요시 자동 배치 가능    |

### 대상 영역

- **외부 8영역:** 획득경험치, 크리티컬 확률, 보스 데미지, 일반 데미지, 버프 지속시간, 방어율 무시, 크리티컬 데미지, 상태이상 내성
- **내부 8영역:** STR, DEX, INT, LUK, HP, MP, 공격력, 마력

## F4. 자동 배치 탐색

| 항목 | 내용                              |
| ---- | --------------------------------- |
| 입력 | 블록 목록 + 목표 칸 수 + 우선순위 |
| 출력 | 배치 결과 (1개)                   |

### 최적해 정의

> **0순위 영역 100% 충족 + 목표 칸 수 정확히 일치 + 나머지는 우선순위 순서대로 최대한 채움**

### 배치 규칙

- 모든 블록 상하좌우 연결 (하나의 덩어리)
- 블록 기준점이 중앙 4칸 중 1칸에 위치해야 함
- 회전/반전 자유
- 블록은 항상 통째로 배치

### 동작

- 우선순위 높은 영역부터 채우기 시도
- 탐색 중 로딩 표시
- 사용자 중단 가능

### 버튼

- 배치 시작
- 중단

### 차선책 (0순위 100% 불가 시)

- 어떤 영역이 부족한지 표시
- 가능한 최선의 배치 제안
- 사용자가 목표 칸 수 또는 우선순위 조정

## F5. 대안 추천 _(MVP 제외 - 2차 고도화)_

| 항목 | 내용             |
| ---- | ---------------- |
| 입력 | 배치 불가능 판정 |
| 출력 | 대안 배치 (1개)  |

### 동작

- 우선순위 낮은 영역부터 칸 조절
- 조절한 칸을 더 낮은 순위 또는 내부 영역으로 이동
- 0순위 영역은 유지
- 현재 시도 중인 칸 분배 실시간 표시

---

## F6. 결과 시각화

| 항목 | 내용             |
| ---- | ---------------- |
| 입력 | 배치 결과        |
| 출력 | 유니온 판 시각화 |

### 동작

- 유니온 판에 배치 결과 표시
- 블록별 색상 + 테두리 구분
- 블록 형태만 구별
- 각 영역별 채워진 칸 수 표시

# 기능 명세서

## F1. 블록 입력

### F1.1 API 조회 캐릭터 검색

| 항목 | 내용                                                        |
| ---- | ----------------------------------------------------------- |
| 입력 | 캐릭터 닉네임                                               |
| 출력 | 해당 유니온 캐릭터 목록 (직업, 레벨, 블록 모양)             |
| 예외 | 닉네임 조회 실패 시: 에러 표시 + 재시도 또는 수동 입력 안내 |

### F1.2 캐릭터 선택

| 항목    | 내용                                     |
| ------- | ---------------------------------------- |
| 입력    | 캐릭터 목록                              |
| 출력    | 선택된 캐릭터들의 블록 카운팅            |
| 동작    | 카드 형태로 표시, 클릭으로 선택/해제     |
| 제한    | 유니온 등급에 따른 최대 인원 (최대 45명) |
| 버튼    | 전체 선택 (레벨 높은 순), 전체 해제      |
| 필터    | 등급별 (B/A/S/SS/SSS)                    |
| 메이플M | 별도 슬롯 1명 추가 가능                  |

### F1.3 수동 입력

| 항목 | 내용                    |
| ---- | ----------------------- |
| 입력 | 블록 종류별 개수        |
| 출력 | 블록 카운팅 + 총 칸 수  |
| 동작 | 등급 × 직업별 개수 입력 |

## F2. 영역별 칸 수 입력

| 항목 | 내용                         |
| ---- | ---------------------------- |
| 입력 | 각 영역에 배치할 칸 수       |
| 출력 | 영역별 칸 수 설정 + 총 칸 수 |

### 영역 목록

| 구분 | 영역                                                                                                            | 최대 칸 수 |
| ---- | --------------------------------------------------------------------------------------------------------------- | ---------- |
| 외부 | 획득경험치, 크리티컬 확률, 보스 데미지, 일반 데미지, 버프 지속시간, 방어율 무시, 크리티컬 데미지, 상태이상 내성 | 각 40칸    |
| 내부 | STR, DEX, INT, LUK, HP, MP, 공격력, 마력                                                                        | 각 15칸    |

### 동작

- 각 영역별 칸 수 직접 입력 (0 ~ 최대)
- 0칸 = 미지정 (배치 금지)
- 총 칸 수 실시간 표시

### 유효 스탯 표시

| 입력 방식   | 표시                                     |
| ----------- | ---------------------------------------- |
| 닉네임 검색 | 해당 캐릭터 직업의 유효 스탯만 강조 표시 |
| 수동 입력   | 모든 내부 스탯 동일하게 표시             |

### 제한

- 영역별 최대 칸 수 초과 불가
- 총 칸 수 ≤ 선택한 블록 총합
- 총 칸 수 = 블록 총 칸 수 (남는 칸은 연결용으로 자동 배치)

### 버튼

- 전체 초기화 (모든 영역 0칸)
- 프리셋 적용 (사냥/보스)

### 예외

- 블록 총합 초과 시 에러 표시

## F3. 우선순위 설정

| 항목 | 내용                         |
| ---- | ---------------------------- |
| 입력 | 프리셋 선택 또는 커스텀 설정 |
| 출력 | 영역별 우선순위              |

### 프리셋

| 프리셋 | 0순위                | 1순위                | 2순위            |
| ------ | -------------------- | -------------------- | ---------------- |
| 사냥   | 획득 경험치 40칸     | 크리티컬 데미지 40칸 | 일반 데미지 40칸 |
| 보스   | 크리티컬 데미지 40칸 | 보스 데미지 40칸     | 방어율 무시 40칸 |

### 커스텀

| 항목       | 내용                                  |
| ---------- | ------------------------------------- |
| UI         | 칸반 형태, 드래그로 순서 변경         |
| 0순위      | 필수 영역 (지정 칸 수 100% 채움)      |
| 1순위 이하 | 중요도 순서 (지정 칸 수 최대한 채움)  |
| 미지정     | 배치 금지 (칸 수 입력 불가, 0칸 고정) |

### 제약 조건

| 제약             | 내용                   |
| ---------------- | ---------------------- |
| 0순위 개수       | 최대 3개               |
| 외부 영역 미지정 | 배치 금지              |
| 내부 영역 미지정 | 연결용으로만 배치 허용 |

### 검증

- 설정 완료 시 배치 가능 여부 사전 검증
- 불가능 시 경고 메시지 표시

### 동작

- 영역을 드래그하여 우선순위 그룹 간 이동
- 우선순위 그룹 내에서는 동일 순위
- 미지정으로 이동 시 해당 영역 칸 수 0으로 초기화

## F4. 자동 배치 탐색

| 항목 | 내용                                |
| ---- | ----------------------------------- |
| 입력 | 블록 목록 + 영역별 칸 수 + 우선순위 |
| 출력 | 배치 결과 (1개)                     |

### 최적해 정의

> **0순위 영역 지정 칸 수 100% 충족 + 1순위 이하 영역 지정 칸 수 최대한 충족 + 미지정 영역 0칸**

### 배치 규칙

- 모든 블록 상하좌우 연결 (하나의 덩어리)
- 최소 1개 블록의 기준점이 중앙 4칸 중 하나에 위치
- 회전/반전 자유
- 블록은 항상 통째로 배치
- 미지정 영역에는 배치 불가

### 동작

- 0순위 영역부터 채우기 시도
- 탐색 중 로딩 표시
- 사용자 중단 가능

### 버튼

- 배치 시작
- 중단

### 차선책 (0순위 100% 불가 시)

- 어떤 영역이 부족한지 표시
- 가능한 최선의 배치 제안
- 사용자가 영역별 칸 수 또는 우선순위 조정

## F5. 대안 추천 _(MVP 제외 - 2차 고도화)_

| 항목 | 내용             |
| ---- | ---------------- |
| 입력 | 배치 불가능 판정 |
| 출력 | 대안 배치 (1개)  |

### 동작

- 우선순위 낮은 영역부터 칸 수 조절
- 조절한 칸을 더 낮은 순위 영역으로 이동
- 0순위 영역은 유지
- 현재 시도 중인 칸 분배 실시간 표시

---

## F6. 결과 시각화

| 항목 | 내용             |
| ---- | ---------------- |
| 입력 | 배치 결과        |
| 출력 | 유니온 판 시각화 |

### 동작

- 유니온 판에 배치 결과 표시
- 블록별 색상 + 테두리 구분
- 각 영역별 목표 칸 수 vs 실제 배치 칸 수 표시
- 미충족 영역 강조 표시

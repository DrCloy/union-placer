# 요구사항 정의서

## 1. 기능적 요구사항

### 1.1 블록 입력

#### 1.1.1 API 조회 캐릭터 검색

- 캐릭터 닉네임 입력
- 해당 캐릭터가 등록된 유니온의 캐릭터 목록 조회
- API Key 입력 옵션 (선택)
- 사용자 Key 있으면 사용, 없으면 서비스 Key 사용

#### 1.1.2 수동 입력

- API 없이 블록 종류별 개수 직접 입력 가능

#### 1.1.4 캐릭터 선택

- 월드 단위로 캐릭터 목록 표시
- 다중 선택 가능 (카드 형태 UI)
- 선택 시 블록 종류별 자동 카운팅

### 1.2 영역별 칸 수 입력

- 각 영역(외부 8개 + 내부 8개)에 배치할 칸 수 직접 입력
- 영역별 최대 칸 수 제한 (외부: 40칸, 내부: 15칸)
- 총 칸 수 = 선택한 블록 총합 이하
- 미지정 영역: 배치 금지 (0칸 고정)
- 0으로 초기화 버튼
- 프리셋 적용 버튼

### 1.3 우선순위 설정

#### 1.3.1 프리셋

- 사냥: 획득 경험치 40칸 > 크리티컬 데미지 40칸 > 일반 데미지 40칸
- 보스: 크리티컬 데미지 40칸 > 보스 데미지 40칸 > 방어율 무시 40칸

#### 1.3.2 커스텀

- 사용자가 영역별 칸 수 + 우선순위 직접 설정
- 0순위: 필수 영역 (지정 칸 수 100% 채움)
- 1순위 이하: 중요도 순서 (지정 칸 수 최대한 채움)
- 미지정: 배치 금지

#### 1.3.3 시스템 제약 조건

- 0순위 영역: 최대 3개까지 지정 가능
- 외부 영역 미지정 시: 배치 금지
- 내부 영역 미지정 시: 연결용으로만 배치 허용
- 설정 완료 시 배치 가능 여부 사전 검증

### 1.4 자동 배치 탐색

#### 1.4.1 배치 규칙

- 모든 블록은 상하좌우로 연결되어야 함 (하나의 덩어리)
- 최소 1개 블록의 기준점이 중앙 4칸 중 하나에 위치해야 배치 가능
- 블록은 회전/반전 가능
- 블록은 항상 통째로 배치 (부분 배치 불가)
- 미지정 영역에는 배치 불가

#### 1.4.2 최적해 정의

> **0순위 영역 지정 칸 수 100% 충족 + 1순위 이하 영역 지정 칸 수 최대한 충족 + 미지정 영역 0칸**

#### 1.4.3 탐색

- 영역별 칸 수 + 우선순위 기반 최적 배치 탐색
- 결과는 1개만 표시
- 로딩 표시

#### 1.4.4 차선책

- 0순위 영역을 100% 채우지 못하는 경우
- 어떤 영역이 부족한지 표시
- 가능한 최선의 배치 제안

### 1.5 결과 시각화

- 유니온 판에 배치 결과 표시
- 블록 구분: 색상 + 테두리

## 2. 비기능적 요구사항

### 2.1 성능

- 탐색 시간 제한 없음 (목표: 수 초 ~ 1-2분)
- 사용자가 탐색 중단 가능

### 2.2 호환성

- 모던 브라우저 전체 지원 (IE 제외)
- 모바일 웹 접속 대응

### 2.3 오프라인

- 선택 사항 (우선순위 낮음)

## 3. MVP 범위

### 3.1 MVP (1차)

- 블록 입력 (API 조회 + 수동 입력)
- 영역별 칸 수 입력
- 우선순위 설정 (프리셋 + 커스텀)
- 자동 배치 탐색
- 결과 시각화
- 배치 불가 시: 에러 표시 + 부족한 영역 안내

### 3.2 고도화 (2차)

- 자동 대안 추천 (우선순위 낮은 영역부터 자동 조절)
- 여러 대안 비교 기능
- 배치 결과 저장/불러오기

## 4. 유니온 판 구조

| 항목      | 값                                                                                                              |
| --------- | --------------------------------------------------------------------------------------------------------------- |
| 전체 크기 | 22 × 20 (440칸)                                                                                                 |
| 내부 크기 | 15칸 × 8구역 (120칸)                                                                                            |
| 외부 크기 | 40칸 × 8구역 (320칸)                                                                                            |
| 외부 영역 | 획득경험치, 크리티컬 확률, 보스 데미지, 일반 데미지, 버프 지속시간, 방어율 무시, 크리티컬 데미지, 상태이상 내성 |
| 내부 영역 | STR, DEX, INT, LUK, HP, MP, 공격력, 마력                                                                        |

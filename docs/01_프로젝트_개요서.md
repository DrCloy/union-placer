# 프로젝트 개요서

## 1. 기본 정보

- **프로젝트명 (영문):** union-placer
- **프로젝트명 (한글):** 유니온 최적화 배치기
- **플랫폼:** 웹 서비스

## 2. 목적

기존 유니온 배치기는 "자동"이라고 하면서도 사용자가 직접 칸을 선택해야 한다.
진짜 자동 배치기를 만들자.

## 3. 목표 사용자

- **전체:** 메이플스토리 유저
- **주 타겟:** 유니온 8000 이상 유저

## 4. 차별점

사용자가 일일이 배치될 칸을 눌러가며 배치를 찾지 않아도 된다.
몇 칸을 배치할 것인지만 입력하면 자동으로 최적의 유니온 배치를 찾고,
불가능할 경우 우선순위 기반으로 높은 항목을 최대한 유지하는 배치를 추천해준다.

## 5. 핵심 기능

### 5.1 블록 입력

- Nexon API로 캐릭터 목록 자동 조회
- 캐릭터별 직업/레벨 → 블록 모양 자동 매핑
- 배치할 캐릭터 선택
- 블록 종류별 카운팅

### 5.2 목표 칸 수 입력

- 원하는 총 배치 칸 수 입력

### 5.3 우선순위 설정

- 프리셋: 사냥 / 보스
- 커스텀: 사용자 직접 설정

### 5.4 자동 배치 탐색

- 목표 칸 수 + 우선순위 기반 최적 배치 탐색
- 불가능 시 대안 추천

### 5.5 결과 시각화

- 유니온 판에 배치 결과 표시
- 어떤 블록이 어디에 배치되는지 표시

## 6. 기술 스택

- **프레임워크:** Vite + React
- **언어:** TypeScript
- **스타일링:** Tailwind CSS
- **상태 관리:** Zustand
- **UI:** Stitch로 직접 구현
- **API 통신:** 추후 결정
